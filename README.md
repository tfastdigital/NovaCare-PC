## 🚀 Overview

It combines everyday PC maintenance with advanced diagnostics, security monitoring, and a built-in AI assistant for users who want a single place to clean, optimize, troubleshoot, and understand their system.

## ✨ Comprehensive Feature List

### 📊 Dashboard & System Monitoring
* **Performance Score:** Evaluates system status and identifies user profiles (e.g., "Programmer user").
* **Real-Time Metrics:** Displays System Drive capacity, RAM Usage, active Processes, and system Uptime.
* **Full System Information:** Details Processor architecture, generation, core/thread count, max clock speed, and cache.
* **Monitoring Tools:** Includes a RAM/VRAM Monitor, Hardware Diagnostics, and System Logs.

### 🧹 Cleaning & Storage Management
* **Disk Cleanup:** Scans and removes Temporary Files, Browser Cache, Windows Update Cleanup files, Thumbnail Cache, and Log Files.
* **Duplicate Finder:** Searches specific folders for duplicate files, with filters for Audio, Video, Images, Archives, and Documents.
* **Storage Tools:** Includes a Storage Analyzer and a Disk Health Monitor.

### ⚡ Optimization & Performance
* **RAM & Process Manager:** Lists running apps with their PID and memory usage, allowing users to kill non-essential processes or use a 1-click "Optimize RAM" function. Protects system-critical processes from being closed.
* **Startup Manager:** Enables or disables programs that run at boot to improve startup times.
* **Windows Optimizer:** Applies system tweaks such as disabling transparency effects, animations, smooth scrolling, Windows search indexing, Prefetch/Superfetch, and Hibernation. Can also set High Performance Power Plans and run SFC/DISM repairs.
* **Bloatware Remover:** Detects pre-installed Windows apps and highlights which are safe to remove.
* **Additional Optimizers:** Features an Internet Optimizer, Game Compatibility tool, and an FPS Optimizer.

### 🛡️ Security, Privacy & Protection
* **Security Center:** Monitors the health of Windows Defender, Windows Firewall, and User Account Control.
* **Threat Scanner & Real-Time Monitor:** Detects issues such as modified Windows hosts files.
* **Privacy Tools:** Includes a Privacy Cleaner, Registry Cleaner, Network Blocker, and Windows Update Blocker.
* **Shortcut Virus Remover:** Detects and removes shortcut viruses from drives.

### 🛠️ Advanced & Specialized Utilities
* **PC Buyer Audit:** Specialized tool for auditing hardware when purchasing a PC.
* **Fake Drive Detection:** Capable of detecting fake or spoofed flash drives.
* **NovaCare AI Assistant:** Integrated chat interface (supporting models like GPT-4o Mini) that analyzes the system, provides recommendations, and features an auto-execution toggle.
* **System Managers:** Includes an App Uninstaller, Browser Manager, Screen Tools, and a Windows Features Manager.
* **Quick Actions:** 1-click access to flush the DNS cache, run a quick clean, or empty the recycle bin.

## 💻 System Requirements
